## Launch the App

Install Docker and Docker Compose

- Run all the services: `docker-compose up`
- API: `docker-compose up api`
- App: `docker-compose up app`
- Build: `docker-compose up build`

Version:

- Angular 13.2.0
- Nestjs 8.0.0
- Nx 13.8.3
